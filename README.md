# School Flow

Система управления школьными процессами.

## Описание

School Flow — веб-приложение для автоматизации управления 
образовательным процессом, ведения школьных дневников, контроля и начисления оценок.

Платформа обеспечивает:

- Аутентификацию и авторизацию пользователей
- Управление ролями (Администратор, Учитель, Ученик, Родитель)
- Профили пользователей с возможностью расширенной информации
- Систему приглашений новых пользователей

## Статус

Проект находится в стадии активной разработки.

## Технологии

- Django 5.2
- Python 3.13+
- PostgreSQL
